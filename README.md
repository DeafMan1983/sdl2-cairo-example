# sdl2-cairo-example
A simple SDL2 and cairo example.

## Build
```sh
make
```

## Usage
```sh
./sdl2-cairo
```

## Screenshot
![img](screenshot.png)

## Platform
SDL2 is a cross-platform library.
It has been confirmed that `sdl2-cairo` runs on following platforms.

- Ubuntu
- macOS
- MSYS2 + MinGW-w64 (Windows 10)
